Build an app for predicting house prices by Niels Rozeboom

  1. Home
  2. Blog Home 2

Posted on July 26., 2018 at 14:00 PM

With this application you can predict what the house prices were in Boston back in the 1970s. The prediction is based on a deep learning model that was being trained with the Boston house price dataset.

This dataset comes with the keras library that we will use for this example. Please have a look at the contents of the dataset. There are a total of 506 rows of data with 13 columns. Every row has one output value and that is the price of the house.

To start working with this dataset we have to load this dataset from KERAS. If you want to do this, just copy the 2 lines of code you see here below. As you might see, the dataset is getting split in a train and test dataset. The train dataset is then a subset which we use to train the model. The test dataset is a smaller subset we will use to test the predictions of the model.

So far so good and straightforward. Now it is getting a little bit tricky, because we have to prepare the dataset so we can use it to create our model. So what do we have to do? We have to normalize the data first. And what does that mean? That means that all features have more or less the same range so that each feature contributes approximately proportionately. If we would leave out this step a feature with a broad range of values will probably dominate the final outcome. It is nice that python allows us an uncomplicated notation of this normalization step.
